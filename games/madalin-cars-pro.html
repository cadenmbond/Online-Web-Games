Controls
WASD / arrow keys = drive around
L-shift = use nitro
Q = change image quality
R = reset car's position
Y = restart game
T = show map
F = full screen
G = change auto / manual gear
H = turn on / off police light
J = change car's engine sound
C = change camera view
B = show center rearview mirror
M (2x) = go to main menu
Space = hand brake
